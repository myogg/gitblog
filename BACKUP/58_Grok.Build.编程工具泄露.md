# [Grok Build 编程工具泄露](https://github.com/myogg/gitblog/issues/58)

Grok Build 是一款跨平台 AI Agent 工作流应用，主打自主执行多步开发任务，不用人工一步步指令，就能完成项目规划、编码、调试、部署全流程。
 
![IMG_20260510_213749_525.jpg](https://i.829259.xyz/api/rfile/IMG_20260510_213749_525.jpg)

工具默认搭载 Grok 4.3 Early Access，权限给到非常激进：开放本地文件全访问、Git 仓库操作权限，同时支持 MCP 模型上下文协议、官方技能扩展以及第三方插件，适配日常开发、项目协作、代码管理等各类场景。
 
早在之前，马斯克就公开放话：今年6月将推出编程能力全面超越 Claude 的全新大模型。
 
这次泄露信息更是实锤了 xAI 的野心布局：团队正在同时训练多款超大参数规模模型，包含 1万亿、1.5万亿、6万亿、10万亿参数 版本，还有面向图像视频的多模态模型 Imagine V2，全赛道补齐能力。
 
按照行业标准，想要达到 Claude Opus 级别、对标 Claude Code 的顶尖编程推理能力，至少需要6万亿参数。不难看出，xAI 早已精准卡位，6万亿参数模型就是专门用来正面抗衡 Claude 旗舰能力。
 
一边是生态成熟、稳居第一梯队的 Claude Code，一边是马斯克带队、携万亿参数大模型+专属编程Agent工具入局的 xAI。
 
今年6月，AI 编程领域注定要有一场巅峰对决，就看 xAI 能不能兑现承诺，真正实现编程能力反超。
