# [PHP 项目淘汰 PHP 许可证](https://github.com/myogg/gitblog/issues/52)

PHP 项目正式宣布淘汰 PHP 许可证，切换到 3-Clause BSD License。

PHP 许可证属于与 GPL 不兼容的自由软件许可证，因为许可证限制了对“PHP”一词的使用。

该许可证也赋予 PHP Group 修改许可证的权力，而修改许可证需要获得每一位 PHP Group 创始成员的书面同意。

PHP 项目包含了由 Zend Technologies 开发的 Zend Engine，Zend Technologies 于 2019 年被 Perforce Software 收购，Perforce 也已经同意了许可证更改。

PHP 项目宣布他们已经获得了修改许可证的完整授权。
